## Development
```
# download the repo
git clone https://github.com/etaiklein/marukibot.git
# install dependencies
cd marukibot
npm install
npm install -g nodemon
```
Grab token from discord or heroku, and update the .env

```
# run with nodemon to pick up local changes
nodemon
```

## Deployment
